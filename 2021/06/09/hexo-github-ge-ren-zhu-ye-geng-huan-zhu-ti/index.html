hexo+GitHub个人主页更换主题

博客 hexo
hexo
发布日期:   2021-06-09
文章字数:   962
阅读时长:   3 分
阅读次数:  

序言

上一篇博客我为大家介绍了如何使用hexo+GitHub搭建个人博客，本片我将记录我如何更换hexo的主题matery。

主题的下载与切换

下载

hexo主题的官网：https://hexo.io/themes/

matery主题官网：https://github.com/blinkfox/hexo-theme-matery

可以在官网的release里面下载，也可以使用git clone https://github.com/blinkfox/hexo-theme-matery.git 下载，将下载的文件夹放到博客文件夹下的themes下。

切换

在博客文件夹下，打开_config.yml文件，将theme设置为你的主题名字。

主题的配置

增加页面

增加tags标签页、categories分类页、about关于我页面、contact留言板页。

新建分类 categories 页

categories 页是用来展示所有分类的页面，如果在你的博客 source 目录下还没有 categories/index.md 文件，那么你就需要新建一个，命令如下：

hexo new page "categories"

编辑你刚刚新建的页面文件 /source/categories/index.md，至少需要以下内容：

---
title: categories
date: 2021-06-8 17:25:30
type: "categories"
layout: "categories"
---

新建标签 tags 页

tags 页是用来展示所有标签的页面，如果在你的博客 source 目录下还没有 tags/index.md 文件，那么你就需要新建一个，命令如下：

hexo new page "tags"

编辑你刚刚新建的页面文件 /source/tags/index.md，至少需要以下内容：

---
title: tags
date: 2021-06-8 17:23:38
type: "tags"
layout: "tags"
---

新建关于我 about 页

about 页是用来展示关于我和我的博客信息的页面，如果在你的博客 source 目录下还没有 about/index.md 文件，那么你就需要新建一个，命令如下：

hexo new page "about"

编辑你刚刚新建的页面文件 /source/about/index.md，至少需要以下内容：

---
title: about
date: 2021-06-8 17:25:30
type: "about"
layout: "about"
---

新建留言板 contact 页

contact 页是用来展示留言板信息的页面，如果在你的博客 source 目录下还没有 contact/index.md 文件，那么你就需要新建一个，命令如下：

hexo new page "contact"

编辑你刚刚新建的页面文件 /source/contact/index.md，至少需要以下内容：

---
title: contact
date: 2021-06-8 17:25:30
type: "contact"
layout: "contact"
---

注：本留言板功能依赖于第三方评论系统，请激活你的评论系统才有效果。并且在主题的 _config.yml 文件中，第 19 至 21 行的“菜单”配置，取消关于留言板的注释即可。

新建 404 页

如果在你的博客 source 目录下还没有 404.md 文件，那么你就需要新建一个

hexo new page 404

编辑你刚刚新建的页面文件 /source/404/index.md，至少需要以下内容：

---
title: 404
date: 2021-06-8 17:25:30
type: "404"
layout: "404"
description: "Oops～，我崩溃了！找不到你想要的页面 :("
---

增加emoji支持

:smile：使你的markdown博客里的emoji表情跳动起来

安装插件：

npm install hexo-filter-github-emojis --save

在博客文件夹下增加下面配置：

githubEmojis:
  enable: true
  className: github-emoji
  inject: true
  styles:
  customEmojis:

代码高亮

如果你的博客中曾经安装过 hexo-prism-plugin 的插件，那么你须要执行 npm uninstall hexo-prism-plugin 来卸载掉它，否则生成的代码中会有 &#123; 和 &#125; 的转义字符。

然后，修改 Hexo 根目录下 _config.yml 文件中 highlight.enable 的值为 false，并将 prismjs.enable 的值设置为 true，主要配置如下：

highlight:
  enable: false
  line_number: true
  auto_detect: false
  tab_replace: ''
  wrap: true
  hljs: false
prismjs:
  enable: true
  preprocess: true
  line_number: true
  tab_replace: ''

主题中默认的 prismjs 主题是 Tomorrow Night，如果你想定制自己的主题，可以前往 prismjs 下载页面 定制下载自己喜欢的主题 css 文件，然后将此 css 主题文件取名为 prism.css，替换掉 hexo-theme-matery 主题文件夹中的 source/libs/prism/prism.css 文件即可。
