剑指offerJZ12数值的整数次方题解

描述

给定一个double类型的浮点数base和int类型的整数exponent。求base的exponent次方。

保证base和exponent不同时为0。不得使用库函数，同时不需要考虑大数问题，也不用考虑小数点后面0的位数。

示例1

输入：2.00000,3

返回值：8.00000

示例2

输入：2.10000,3

返回值：9.26100

示例3

输入：2.00000, -2

返回值：0.25000

暴力法

不断进行乘法运算即可。

public class Solution {
    public double Power(double base, int exponent) {
        double ans = 1.0;
        if (exponent == 0){
            return ans;
        } else if (exponent > 0){
            ans = base;
            for (int i = 0; i < exponent - 1; i++) {
                ans *= base;
            }
        } else {
            ans = 1 / base;
            double temp = ans;
            for (int i = exponent + 1; i < 0; i++){
                ans *= temp;
            }
        }
        return ans;
    }
}

递归法（快速幂）

算法思路：

  1. 如果是exponent是负数，就先转换，然后进入递归方法
  2. 在递归方法中进行判断，如果exponent为 0，则返回 1.0
  3. 判断exponent为技术还是偶数，如果是奇数，只需将base留出一个，继续进入递归
public class Solution {
    public double Power(double base, int exponent) {
        if (exponent < 0){
            base = 1 / base;
            exponent /= -1;
        }
        return pow(base, exponent);
    }

    public double pow(double base, int exponent){
        if (exponent == 1){
            return base;
        } else if (exponent == 0){
            return 1.0;
        } else if (exponent == 2){
            return base * base;
        }
        if (exponent % 2 == 1){
            return base * pow(base, exponent - 1);
        } else {
            return pow(pow(base, exponent / 2), 2);
        }
    }
}

非递归法（快速幂）

算法思路：

  • $2.0^{7}=2.0^{1+2+4}$，7的二进制位为：0111
  • 按这样的思路，我们就可以只对指数二进中为 1 的位进行运算
public class Solution {
    public double Power(double base, int exponent) {
        if (exponent < 0){
            base = 1 / base;
            exponent /= -1;
        } else if (exponent == 0) {
            return 1.0;
        }
        double res = 1.0;
        while (exponent != 0){
            if ((exponent & 1) != 0){
                res *= base;
            }
            base *= base;
            exponent >>= 1;
        }
        return res;
    }
}
